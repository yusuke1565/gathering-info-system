# num_file
　このディレクトリは、ユーザの実験回数やユーザIDなどの情報を格納している場所である。特に手を加える必要はないが、誰がどの問題まで行ったなど、確認ができる。

### IDnum.txt
　このファイルには、次にアクセスする新しいユーザに割り振られる、IDの値を記している。最初は0から始まり、IDが割り振られるごとに、+1 されていく。

### of_times.txt
 このファイルには、誰が何回目の実験かを記している。2と記されていれば、既に2回実験を行っていることになる。1行目にはIDが0の人の情報が、2行目にはIDが1の人の情報が記されている。

### start_position_prob.txt 
　このファイルには、誰が何問目まで解答したかを記している。１行目にはIDが0の人の情報が、2行目にはIDが1の人の情報が記されている。<br>
　例えば、4行目に7と記されていたならば、IDが3の人は次回は8問目からの解答になる。
