# templates
　このディレクトリには、このシステムで使用する、HTMLファイルが置かれている。

### one/two/three/four/five.html
　これらのHTMLファイルは、Phaseごとのファイルです。Phaseについては、[README.md](../README.md)を参照してほしい。

### その他のファイル
　"index.html"は、ホームのページである。<br>
　"ready.html"は、ホームで"初めて"のボタンを押すと飛ぶ、準備画面のページである。<br>
　"input_userID.html"は、ホームで"2回目以降"のボタンを押すと飛ぶ、ID入力画面のページである。<br>
　"ex_end.html"は、例題が終了した時に表示されるページである。<br>
　"thanks.html"は、すべての問題が終了した時に表示されるページで、実験終了と、実験参加への感謝の旨を伝えるページである。
